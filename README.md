# 🧠 GI Disease Detection System

This project is a deep learning-based web application that detects gastrointestinal diseases such as **Polyp, Ulcer, and Normal** using medical images.

---

## 🚀 Features

* Image upload for disease prediction
* Detects Polyp, Ulcer, and Normal conditions
* Fast and simple web interface
* Deep learning-based classification

---

## 🛠️ Tech Stack

* Python
* Flask
* TensorFlow / Keras
* HTML, CSS

---

## ▶️ How to Run

1. Clone the repository
   git clone https://github.com/syed-kashif-245/context-aware-firewall.git

2. Go to project folder
   cd context-aware-firewall

3. Install dependencies
   pip install -r requirements.txt

4. Run the app
   python app.py

5. Open in browser
   http://127.0.0.1:5000/

---
